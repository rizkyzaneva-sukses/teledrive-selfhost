Panduan penggunaan

TeleDrive Selfhost

Web app ini dipakai sebagai file manager rapi. File fisik tetap tersimpan di Telegram Channel, sementara folder, tag, search, preview, dan trash dikelola dari web app.

Buka Web App

Konsep Utama

Telegram Channel

Berfungsi sebagai gudang fisik file. Kalau dibuka langsung dari Telegram, file memang bisa terlihat campur.

Web App

Berfungsi sebagai rak dan label. Di sini file terlihat rapi dalam folder, tag, search, recent, trash, dan activity.

Metadata

Nama file, folder, tag, ukuran, tanggal, dan ID pesan Telegram disimpan oleh web app supaya file mudah dicari.

Workflow Harian

  1. Buka web app dan masukkan admin token.
  2. Masuk ke folder tujuan, atau buat folder baru.
  3. Pilih satu atau banyak file sekaligus.
  4. Tambahkan tag bila perlu, contoh: produk,konten,penting.
  5. Klik upload. File fisik masuk ke Telegram Channel, tampilan rapi muncul di web app.
  6. Untuk mencari file, gunakan search, filter tipe, Recent, tag, atau sort.
  7. Untuk melihat cepat, gunakan Preview atau Gallery.

Fitur Utama

Multi Upload

Pilih banyak file, lalu upload semuanya ke folder aktif.

Folder Drive

Klik folder untuk masuk. Upload selalu masuk ke folder yang sedang dibuka.

Rename & Move

Gunakan tombol Edit di file untuk rename, pindah folder, atau update tag.

Search & Sort

Cari nama file, folder, tipe file, tanggal, atau tag. Urutkan berdasarkan terbaru, nama, ukuran, atau tipe.

Gallery & Preview

Gallery cocok untuk foto produk. Preview mendukung gambar, video, audio, dan PDF.

Trash & Restore

File yang dihapus dari web masuk Trash dulu dan bisa direstore.

Catatan Penting

Jangan jadikan Telegram Channel sebagai tempat kerja harian. Pakai web app untuk cari, preview, dan download.

Delete dari web app menghapus file dari tampilan utama dan memindahkan metadata ke Trash. File fisik di Telegram Channel tidak ikut terhapus.

DOCX/XLSX biasanya tidak bisa preview langsung seperti PDF di browser. Gunakan Open atau Download.

Activity log mulai lengkap untuk aktivitas setelah fitur activity dibuat. Aktivitas lama tidak bisa lengkap secara retroaktif.

Kalau tampilan belum berubah, refresh browser atau buka tab incognito karena browser HP sering menyimpan cache lama.
